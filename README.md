# comfyui-textarea-command

Add command when cursur in textarea.

## Usage  

| Command      | Result               |
|--------------|----------------------|
| \"            | \"...\"                |
| \'            | \'...\'                |
| \`            | \`...\`                |
| \{            | \{...\}                |
| \[            | \[...\]                |
| \(            | \(...\)                |
| Ctrl+\/       | \/\/ This line is disabled... |
| Tab          | Select next word     |
| Shift+Tab    | Select previous word |
| Ctrl+Z       | Undo                 |
| Ctrl+Shift+Z | Redo                 |
